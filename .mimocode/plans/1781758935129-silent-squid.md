# Plan: Fix Battery Info + File Installation

## Root Cause Analysis

The Rust corelib is identical between AstroBox-NG and Miband. The protocol stack works. The issues are in the Kotlin `BandBurgManager`:

1. **Auth handshake** sends AuthStep1 but `handleReceivedPackets` may not correctly parse DeviceVerify from JSON
2. **Battery info** - commands timeout because auth hasn't completed
3. **File installation** - same root cause as battery

## Fix Strategy

### Step 1: Add debug logging to see actual packet format
Add `Log.w(TAG, "Raw packet: $jsonResult")` before parsing in `handleReceivedPackets`

### Step 2: Don't throw on auth timeout
Change `connect()` to proceed even if auth times out (just log warning)

### Step 3: Add retry logic for getDeviceInfo
If first attempt returns empty data, wait 1s and retry once

### Files to modify
- `shared/src/androidMain/kotlin/com/miband/app/core/BandBurgManager.kt`
- `shared/src/commonMain/kotlin/com/miband/app/App.kt`
